# 个人理解

javaScript早起有很多的不足，没有一个统一的标准，为了让其走的更远，所以就有了模块机制（所谓的没有规矩不成方圆）

模块机制的诞生流程 CommonJs => AMD ==> CMD

commonJs 为javaScript 制定了一个美好的愿景 -- 希望javaScript 能够在任何地方运行

commonJs规范的提出，主要是为了弥补当前JavaScript没有标准的缺陷，
