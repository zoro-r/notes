## Jest

Jest是 Facebook提供的一款轻量级 JavaScript :测试框架，它具有特点:

- 开箱即用，配置少，API简单，上手成本极低在沙箱中运行，更加安全
- 支持断言和仿真
- 自动生成测试覆盖率报告
- 通过生成Snapshot 进行 UI 测试单测执行效率，

## Enzyme

- 专门用于React 测试工具，
- 方便操作 Dom 且操作风格模拟了jQuery的APi，比较直观，学习使用都比较简单
- 便利的工具函数库封装，可以处理浅渲染，静态渲染标记以及DOM渲染。
